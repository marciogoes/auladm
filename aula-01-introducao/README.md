## Estrutura dos Exemplos

Cada pasta contém um arquivo `App.js` pronto para ser colado no [snack.expo.dev](https://snack.expo.dev).

| # | Exemplo | Conceito Abordado |
|---|---------|-------------------|
| 01 | `exemplo-01-ola-tads` | Estrutura básica: View, Text, StyleSheet |
| 02 | `exemplo-02-contador` | Interatividade: useState, Button, onPress |
| 03 | `exemplo-03-formulario-alert` | Atividade prática: TextInput, Button, Alert |
| 04 | `exemplo-04-flatlist` | Desafio: FlatList substituindo map() |
| 05 | `exemplo-05-props` | Desafio: Passagem de props e desestruturação |
| 06 | `exemplo-06-asyncstorage` | Desafio: useEffect + AsyncStorage |
| 07 | `exemplo-07-flexbox-layouts` | Estilização: Flexbox no mobile |
| 08 | `exemplo-08-web-vs-mobile` | Comparativo: HTML vs React Native |

## Como Usar

1. Acesse [snack.expo.dev](https://snack.expo.dev)
2. Copie o conteúdo do `App.js` desejado
3. Cole no editor do Snack
4. Escaneie o QR Code com o Expo Go no celular

## Pré-requisitos

- App **Expo Go** instalado no smartphone
- Conta gratuita no snack.expo.dev
- Conhecimento básico de JavaScript ES6+ e React
